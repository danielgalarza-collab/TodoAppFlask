# Todo App - Flask

Aplicación web CRUD con autenticación de usuarios desarrollada con Flask y SQLAlchemy.

## Funcionalidades

- Registro y login de usuarios
- CRUD completo de tareas
- Sistema de sesiones
- Protección de rutas con decoradores
- Prioridad y estado de tareas

## Tecnologías

- Python 3
- Flask
- SQLAlchemy
- SQLite
- PostgreSQL
